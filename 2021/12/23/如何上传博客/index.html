如何上传博客

1、先找到你的blog的文件夹

2、在文件夹中右键 git bash

3、利用 hexo new post XXX xxx是你的博客中文章的名字

4、在博客文件夹下的source的_posts中找到你的.md后缀的文章名字

5、在md文件中写入你的博客

6、利用命令 hexo g生成你文章的静态文件

7、利用命令hexo d上传到GitHub中
